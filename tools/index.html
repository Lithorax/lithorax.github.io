Geoscience Tools

Nine free calculators covering sedimentology, geochemistry, hydrogeology, petrophysics, stratigraphy, and planetary geology. Click any card to open the tool — no login, no download, runs entirely in your browser.

🪨
Sedimentology Tool 01
Grain Size Classifier
Classify sediment grain size using the Wentworth Scale. Accepts input in millimetres, micrometres, or Φ (phi) units. Returns sediment class, depositional environment interpretation, and phi equivalent.
Wentworth Scale mm / μm / Φ Depositional Env.

Geochemistry Tool 02
CIA Calculator
Calculate the Chemical Index of Alteration (CIA) from major oxide weight percentages. Quantifies the intensity of silicate chemical weathering with geological interpretation.
Al₂O₃ · CaO* · Na₂O · K₂O Weathering Intensity
Open Tool →

Geochemistry Tool 03
Geochemical Indices
Compute five major geochemical weathering and maturity indices simultaneously — CIA, PIA, CIW, ICV, and WIP — from a single major oxide dataset.
CIA PIA CIW ICV WIP
Open Tool →

Hydrogeology Tool 04
Darcy's Law Calculator
Solve for groundwater flow rate Q, hydraulic conductivity K, or hydraulic gradient i using Darcy's Law (Q = K × i × A). Includes rock-type conductivity reference.
Flow Rate Q Conductivity K Gradient i

Porosity Calculator

Sedimentology Tool 06
Phi Scale Converter
Bidirectional conversion between millimetres and the phi (Φ) scale using Φ = −log₂(d mm). Includes a full Wentworth grain-size reference table.
mm → Φ Φ → mm Reference Table

Impact Crater Calculator
Estimate final crater diameter from impactor properties using Pi-group scaling relations (Schmidt & Housen 1987; Melosh 1989). Supports Earth, Mars, Moon, and Venus gravity fields.
